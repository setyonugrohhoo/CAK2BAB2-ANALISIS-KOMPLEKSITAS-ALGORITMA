Perhitungan jumlah digit pada suatu bilangan bulat merupakan permasalahan
dasar dalam pemrograman yang sering digunakan untuk memahami konsep

dilakukan menggunakan bahasa pemrograman Python. Dua pendekatan
algoritmik yang digunakan adalah metode iteratif dan metode rekursif.
